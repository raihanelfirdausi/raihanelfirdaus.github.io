Project information

  • Category: Remote Sensing & GIS
  • Short Title: Land Cover Change Analysis and Carbon Loss Estimation : Blora Regency, East Java
  • Affiliation: Personal Project

Description:

This project aims to analyze land cover changes in Blora Regency from 2004 to 2024 and predict future changes up to 2034. The analysis begins with utilizing Google Earth Engine (GEE) to perform land cover classification using the supervised classification method on time-series data. The analysis focuses on three main land cover categories: forest, cropland, and built-up areas. The data is then processed in QGIS to prepare additional parameters, such as a Digital Elevation Model (DEM) converted into slope data, and raster proximity data derived from distances to roads, rivers, and built-up areas. All these datasets are input into the MOLUSCE plugin to model future land cover changes for 2034.

Once the land cover prediction maps are created, the project analyzes pixel changes in each land cover category to illustrate the dynamics of land-use changes over the last 30 years. Furthermore, the project includes carbon storage estimation for each land cover category and calculates the economic value of carbon lost due to land cover changes. These estimations are conducted using the InVEST application, with carbon pool data obtained from official references, such as the Technical Guidelines for Monitoring, Evaluation, and Reporting on the Implementation of Regional Action Plans for GHG Reduction (2013) and the IPSDH results for 1996–2013. The economic value of carbon is based on the carbon price per ton released by Indonesia’s Financial Services Authority (OJK) in August 2024.

The analysis results show a significant carbon loss of -1,829,823.7 tons from 2004 to 2034. This translates to an economic value of $69,350,318.08, equivalent to approximately IDR 112.2 billion. The land cover change visualization and carbon storage estimation were conducted using a combination of tools such as GEE, QGIS, the MOLUSCE plugin, and InVEST to ensure accurate and reliable results. This project highlights the importance of sustainable land management to mitigate carbon loss and its environmental impacts while showcasing the practical application of geospatial technologies in data-driven environmental management.
